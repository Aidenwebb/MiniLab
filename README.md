# Aiden's Homelab

This project uses Infrastructure as Code and GitOps to automate the provisioning of my homelab.

## Overview

### Hardware

- 3x `HP Elitedesk 800 G2`
  - CPU: `Intel I5-6500T @ 2.5 Ghz`
  - RAM: `40GB / 24GB / 16GB` respectively
  - SSD:
    - Boot: `128GB`
    - Storage: `1TB NVMe`
- 1x `TP Link TL-SG2008`
  - Ports: `8`
  - Link Speed: `1000Mbps`
